What Is Calendario?

Calendario is a decentralized, federated protocol that lets anyone publish their availability, preferences, pricing, and booking endpoints via a simple .well-known JSON file. Think of it as a DNS for time ‚Äî declarative, resolvable, and universally linkable.

How It Works

Host a JSON file at: https://yourdomain.com/.well-known/calendario/yourname.json

  • Declare availability, time zone, booking, and pricing logic
  • Include optional organization/identity proofs
  • Share your ID (e.g., @jane@yourdomain.com)
  • Let others resolve it intelligently
